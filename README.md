# docker-awscli

Basic image to use for builds.

This image is based on php 7.2.10 FPM over Alpine linux

## Features

Set of tools to run for building several kind of projects.

Tools in this image:
* aws cli: 1.16.44
* eb: 3.14.6
* php: 7.2.10-fpm
* node: v8.11.4
* npm: 5.6.0
* composer: 1.7.2
* python: 2.7.15
