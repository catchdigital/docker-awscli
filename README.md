# docker-awscli

Basic image to use for builds.

This image is based on php 7.3.14 FPM over Alpine linux

## Features

Set of tools to run for building several kind of projects.

Tools in this image:
* aws cli: 2.0.26
* aws cdk: 1.47.1a
* eb: 3.18.1
* php: 7.4.7-fpm
* node: v12.18.1
* npm: 6.14.5
* composer: 1.10.8
* python: 2.7.16 / 3.7.3
