# docker-awscli

Basic image to use for builds.

This image is based on php 7.3.14 FPM over Alpine linux

## Features

Set of tools to run for building several kind of projects.

Tools in this image:
* aws cli: 2.0.7
* eb: 3.17.1
* php: 7.4.4-fpm
* node: v12.16.1
* npm: 6.13.4
* composer: 1.10.1
* python: 2.7.16
