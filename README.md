# docker-awscli

Basic image to use for builds.

This image is based on php 7.4-fpm

## Features

Set of tools to run for building several kind of projects.

Tools in this image:
* aws cli: 2.2.37
* aws cdk: 1.122.0
* aws amplify: 5.5.0
* eb: 3.20.2
* php: 8.0.10-fpm
* node: v12.22.6
* npm: 6.14.15c
* composer: 2.1.6p
* python: 3.9.2
