# docker-awscli

Basic image to use for builds.

This image is based on php 7.4-fpm

## Features

Set of tools to run for building several kind of projects.

Tools in this image:
* aws cli: 2.2.37
* aws cdk: 1.122.0
* aws amplify: 5.5.0
* eb: 3.20.2c
* php: 8.0.10-fpm
* node: v14.17.6
* npm: 7.23.0
* composer: 2.1.7
* python: 3.9.2
